# Optimisation des Recherches de Veille M&A

## Stratégies de Recherche par Type de Demande

### Recherche d'Opérations Récentes
**Requête type** : "Quelles sont les dernières opérations M&A PME en France ?"
1. CFNEWS → rubrique Fusions-Acquisitions (filtre date)
2. Les Échos → rubrique Finance & Marchés > Fusions-acquisitions
3. BODACC → alertes cessions de fonds de commerce récentes
4. Décideurs Magazine → actualité deals

### Recherche Sectorielle
**Requête type** : "M&A dans le secteur santé en France ce trimestre"
1. CFNEWS → filtre secteur + date
2. France Invest → études thématiques sectorielles
3. Les Échos → recherche par mots-clés sectoriels
4. Cabinets → études sectorielles dédiées (EY, Deloitte)

### Recherche Réglementaire
**Requête type** : "Évolutions réglementaires impactant le M&A en France"
1. AMF → doctrine, positions, recommandations
2. DG Trésor → textes IEF, décrets
3. Ministère Économie → réformes fiscales
4. Ordre EC / CNCC → doctrine comptable et audit

### Recherche de Tendances
**Requête type** : "Tendances M&A mid-market France 2025-2026"
1. France Invest + Grant Thornton → baromètre semestriel
2. EY → baromètre trimestriel transactions
3. In Extenso Finance → panorama PME
4. Bpifrance Le Lab → études prospectives

### Recherche sur une Entreprise Cible
**Requête type** : "Informations sur [Entreprise X] dans un contexte M&A"
1. Pappers → données financières, dirigeants, bénéficiaires effectifs
2. BODACC → modifications récentes (capital, cession, procédure)
3. INPI → brevets, marques (signaux innovation)
4. CFNEWS / presse → historique opérations impliquant cette entreprise

## Mots-Clés Efficaces par Contexte

### M&A Général
`fusion acquisition PME France`, `cession entreprise`, `rachat société`, `opération M&A mid-market`

### LBO / Private Equity
`LBO France PME`, `capital-investissement`, `buyout mid-market`, `build-up`, `secondary buyout`

### Transmission
`transmission entreprise`, `cession PME`, `succession dirigeant`, `reprise entreprise`

### Réglementaire
`contrôle investissements étrangers France`, `notification concentration`, `réforme fiscale cession`, `Dutreil transmission`

### Transaction Services
`due diligence France`, `transaction services`, `vendor due diligence`, `audit acquisition`

## Filtrage Temporel

### Bonnes Pratiques
- Toujours spécifier une période explicite dans la recherche
- Vérifier que la date de l'article correspond bien à la période demandée
- Attention aux articles "mis à jour" qui peuvent afficher une date récente pour du contenu ancien
- Les communiqués de presse ont souvent un décalage de 1-3 jours avec l'opération réelle

### Pièges Fréquents
- Article daté de 2026 qui reprend une opération de 2025 (vérifier la date d'annonce réelle)
- Baromètre publié en T1 couvrant les données de l'année N-1
- Études avec mention "données à fin [date]" vs date de publication
