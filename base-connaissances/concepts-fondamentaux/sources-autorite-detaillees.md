# Sources d'Autorité - Fiches Détaillées

## TIER 1 - RÉGULATEURS & AUTORITÉS

### AMF - Autorité des Marchés Financiers
- **URL** : https://www.amf-france.org
- **Contenu clé** : franchissements de seuils, notes OPA/OPE, communiqués sanctions, doctrine
- **Fréquence MAJ** : quotidienne
- **RSS** : Oui, par catégorie
- **Accès** : Gratuit
- **Rubriques à surveiller** : Actualités > Communiqués de presse ; Réglementation > Doctrine
- **Pertinence M&A PME** : Moyenne (plutôt sociétés cotées, mais doctrinaire applicable)

### ACPR
- **URL** : https://acpr.banque-france.fr
- **Contenu clé** : agréments, sanctions, réglementaire secteur financier
- **RSS** : Oui
- **Accès** : Gratuit
- **Pertinence M&A PME** : Élevée pour acquisitions dans secteur financier/assurance

### Banque de France
- **URL** : https://www.banque-france.fr
- **Contenu clé** : taux directeurs, études sectorielles, observatoire financement entreprises
- **Publication** : Rapports trimestriels et annuels
- **Accès** : Gratuit
- **Pertinence M&A PME** : Élevée (conditions de financement, médiation crédit)

### Ministère de l'Économie
- **URL** : https://www.economie.gouv.fr
- **Contenu clé** : réformes fiscales, contrôle investissements étrangers, aides entreprises
- **RSS** : Oui
- **Accès** : Gratuit
- **Pertinence M&A PME** : Élevée (fiscalité cession, dispositifs Dutreil, IEF)

### DG Trésor
- **URL** : https://www.tresor.economie.gouv.fr
- **Contenu clé** : attractivité, contrôle IEF, rapports sectoriels
- **Accès** : Gratuit
- **Pertinence M&A PME** : Moyenne-élevée (IEF si acquéreur étranger)

### INPI
- **URL** : https://www.inpi.fr
- **Contenu clé** : registre RBE, dépôts marques/brevets
- **API** : Données entreprises disponibles
- **Accès** : Gratuit (API freemium)
- **Pertinence M&A PME** : Moyenne (bénéficiaires effectifs, signaux innovation)

## TIER 1 - INSTITUTIONS PROFESSIONNELLES

### France Invest
- **URL** : https://www.franceinvest.eu
- **Contenu clé** : baromètre semestriel capital-investissement, études thématiques
- **Publication** : Semestrielle + trimestrielle
- **Format** : PDF téléchargeable
- **Accès** : Gratuit (études principales)
- **Pertinence M&A PME** : Très élevée (données de référence marché PE français)

### Bpifrance
- **URL** : https://www.bpifrance.fr / https://lehub.bpifrance.fr
- **Contenu clé** : investissements, études sectorielles (Le Lab), baromètre ETI
- **RSS** : Le Hub uniquement
- **Accès** : Gratuit
- **Pertinence M&A PME** : Très élevée (acteur incontournable du financement PME/ETI)

## TIER 1 - BASES DE DONNÉES PUBLIQUES

### BODACC
- **URL** : https://www.bodacc.fr
- **Contenu clé** : cessions fonds, procédures collectives, modifications capital
- **API** : Gratuite, sans authentification
- **MAJ** : Quotidienne
- **Alertes** : Email quotidien par critère
- **Pertinence M&A PME** : Très élevée (signal d'opérations en temps réel)

### Pappers
- **URL** : https://www.pappers.fr
- **Contenu clé** : 3,6M entreprises, comptes annuels, dirigeants, bénéficiaires effectifs
- **API** : Freemium (100 crédits/mois gratuits), payant à partir de 30€/mois
- **Pertinence M&A PME** : Très élevée (outil de screening et d'analyse cible)

### data.gouv.fr
- **URL** : https://www.data.gouv.fr
- **Contenu clé** : base SIRENE, BODACC historique, RBE INPI
- **API** : Gratuite
- **Pertinence M&A PME** : Élevée (données brutes open data)
