# Changelog - GPT Veille M&A Transaction Services

## Mécanisme de Version
- **MAJEURE (X.0.0)** : Changements structure fondamentale, refonte sources, nouvelle architecture
- **MINEURE (0.X.0)** : Nouveaux patterns/procédures, ajout sources, nouveaux templates
- **PATCH (0.0.X)** : Corrections erreurs, mise à jour liens, ajustements mineurs

---

## [1.0.0] - 2026-02-09

### Création Initiale
**Type :** Version majeure - Lancement du système

### Structure Créée
- `instructions/` : Instructions GPT principales (<7 500 car.), prompt système, directives sécurité
- `base-connaissances/concepts-fondamentaux/` : Glossaire M&A, écosystème France, sources détaillées
- `base-connaissances/procedures/` : Méthodologie veille, workflow analyse opération
- `base-connaissances/meilleures-pratiques/` : Qualité sourcing, optimisation recherche
- `base-connaissances/depannage/` : Problèmes accès sources
- `base-connaissances/ressources/` : Liens RSS/API, templates restitution
- `exemples/` : Conversations exemples, cas d'utilisation
- `versions/` : Changelog, roadmap
- `metriques/` : KPIs, tests validation, feedback
- `apprentissage-continu/` : Patterns succès, erreurs fréquentes, optimisations, évolution comportement
